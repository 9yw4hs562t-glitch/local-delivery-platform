# Local Delivery Platform

Plateforme de livraison locale reliant clients, restaurants/pizzerias et livreurs.

## Principe
Le client commande en ligne, le restaurant prépare, et le livreur le plus proche livre la commande.

## Objectif
Monétisation par abonnement des restaurants et des livreurs.

## Utilisateurs
- Client
- Restaurant / Pizzeria
- Livreur
- Admin

## Fonctionnalités clés
- Commande en ligne
- Menu disponible / non disponible
- Localisation obligatoire (client & livreur)
- Livraison avec suivi

## Portée
Lancement local (wilaya), extension progressive.
